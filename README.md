# Advanced-Number-Guessing-Game
An advanced console-based number guessing game written in C featuring AI opponent mode, multiplayer gameplay, persistent leaderboards, statistics tracking, and interactive terminal UI.

## Features
- Multiple difficulty levels
- AI opponent mode
- Two-player mode
- Leaderboard system
- Persistent score saving
- Statistics dashboard
- Input validation
- Colored terminal UI
- Progress bars
- Hint system

## Technologies Used
- C
- GCC
- File Handling
- ANSI Escape Codes

## How to Run

gcc main.c -o game
./game
