# Advanced Number Guessing Game

A console-based Number Guessing Game developed in C.

## Features
- Random number generation
- Hint system
- Attempt counter
- Difficulty levels
- Replay functionality

## Technologies Used
- C Programming
- Standard Libraries:
  - stdio.h
  - stdlib.h
  - time.h

## How to Run

Compile:
gcc main.c -o game

Run:
./game
